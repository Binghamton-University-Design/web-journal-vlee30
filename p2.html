ARTS 486 : Project 2

Emergent Interaction, Exquisite Corpse

This project called for partners to work together to create graphics while coming up with rules and guidelines for the exquisite corpse. My partner, Evangeline Kontos and I, decided to come up with an environment or landscape theme for the project instead of the traditional body figure of an exquisite corpse. After coming up with our rules, we worked on the graphics and the end result came out to work well together despite the mixing of styles.


Rules for our exquisite corpse:

  • six cells, number order is in Z formation (reading left to right: 1, 2, 4, 3, 5, 6)
  • overall dimensions in pixels:
    1. 1125 x 400
    2. 375 x 400
    3. 750 x 400
    4. 750 x 400
    5. 375 x 400
    6. 1125 x 400
  • landscape/environment theme : sky, land, underwater
  • use the same color palette to create a cohesive look
  • any style of graphics are allowed; Vivienne: odd cells, Evangeline: even cells
  • the design will not be randomized, we will be able to view each other's additions to the exquisite corpse throughout the process
